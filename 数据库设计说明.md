# 数据库设计说明

### article_info 表

| 栏位      | 说明         |
| --------- | ------------ |
| a_title   | 文章题目     |
| a_cid   | 文章分类的id    |
| a_date    | 文章发布日期 |
| a_ldate    | 文章最后修改日期 |
| a_id      | 文章id       |
| a_context | 文章内容     |
| a_view    | 文章浏览次数 |
| a_del     | 删除标志     |

### class_info 表

| 栏位              | 说明           |
| ----------------- | -------------- |
| class_id          | 分类id         |
| class_name        | 分类名字       |
| class_del         | 删除标志       |

### remark_info 表

| 栏位           | 说明       |
| -------------- | ---------- |
| remark_id      | 评论id     |
| remark_aid      | 评论文章的id     |
| remark_name  | 评论者姓名 |
| remark_context | 评论内容   |
| remark_date    | 评论日期   |
| reamrk_email | 评论者邮箱   |
| remark_del    | 删除标志   |
